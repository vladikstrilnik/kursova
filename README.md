# Опис проєкту

Метою проєкту є створення Edge–Cloud системи контролю споживання води в санвузлах торгового комплексу.

Система забезпечує:

- збір даних з імпульсного лічильника води;
- підрахунок кількості спожитої води;
- визначення миттєвої витрати;
- передачу інформації через HTTP API;
- накопичення статистики;
- локальний Web HMI;
- Telegram-сповіщення.
